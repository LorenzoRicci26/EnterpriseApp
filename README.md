# 🎓 University Management System

Sistema gestionale universitario full-stack composto da:

- 🧠 **Backend** in Java Spring Boot con MongoDB
- 🌐 **Frontend** in Angular
- 📦 Deploy con Docker
- ☸️ Orchestrazione con Kubernetes (via Minikube)

---

## ⚙️ Stack Tecnologico

### Backend

- Java 17
- Spring Boot
- MongoDB
- MapStruct
- REST API
- Docker

### Frontend

- Angular
- Angular CLI
- TypeScript
- Docker
